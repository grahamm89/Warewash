How to Use This Tool

Open in Browser: https://grahamm89.github.io/DishWTroubleshoot-BackEnd/ContentsEditor.html

Load Data: Click the "Choose File" button and select your existing app_data.json if you want to edit it. You can download the existing file from this repository. 

The APPS current content will load. 

Edit Content:

Modify the text in any of the fields.

Click the "Add..." buttons to create new entries.

Click the red 'X' on any item to delete it.

Download: When you're finished, click the "Download app_data.json" button. This will save the new file to your computer.

Upload to Host: Finally, upload this newly downloaded file to your web host, replacing the old version. Your live application will now reflect all your changes.
